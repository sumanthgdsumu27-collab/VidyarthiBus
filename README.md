# VidyarthiBus

A student bus tracking and scheduling app for Android.

## About

VidyarthiBus helps college students track their college buses, view routes, and check schedules — all from their Android phone.

## Features

- View bus routes and stops
- Check bus schedules and timings
- Student login and profile
- Bus pass details

## Tech Stack

- Language: Kotlin
- Platform: Android
- Build System: Gradle

## Project Structure

```
VidyarthiBus/
├── app/
│   └── src/
│       ├── main/
│       │   ├── java/        # Kotlin source files
│       │   ├── res/         # Layouts and resources
│       │   └── AndroidManifest.xml
├── build.gradle.kts
├── settings.gradle.kts
└── gradlew
```

## Installation

1. Clone the repository
   ```
   git clone https://github.com/sumanthgdsumu27-collab/VidyarthiBus.git
   ```
2. Open the project in Android Studio
3. Click **Sync Now** when prompted
4. Connect a device or start an emulator and click **Run**

## Build APK

```
./gradlew assembleDebug
```
